# Overview

DeepCare Mobile is an Android application that assists ICU healthcare professionals by providing AI-driven insights to evaluate patient conditions efficiently.

## Key Features

- **AI Predictions**: Mortality risk, length of stay, readmission probability, and key medical timestamps.
- **Clinical Decision Support**: Aggregates multiple clinical inputs for accurate, data-driven recommendations.
- **Optimized UI**: User-friendly interface for quick access to patient data in critical care environments.

## Technical Details

- **Android Development**: Kotlin and Java, following modern architecture patterns.
- **AI Integration**: TensorFlow Lite models run on-device for real-time inference.
- **Data Management**: SQLite for local clinical data; Firebase for authentication and cloud sync.
- **Predictive Accuracy**: Trained on structured clinical datasets, achieving up to 72% accuracy for ICU outcomes.

## Challenges & Solutions

- **Low-Latency Predictions**: On-device AI inference optimized for ICU use.
- **Data Quality**: Preprocessing pipelines handle noisy and incomplete data.
- **Usability**: UI designed to minimize cognitive load for healthcare professionals.

## Outcome

The app supports ICU teams with accurate, AI-powered predictions, improving patient care efficiency and clinical decision-making. Achieved up to 72% prediction accuracy on ICU datasets.
