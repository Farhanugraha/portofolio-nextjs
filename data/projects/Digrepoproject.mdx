# Overview

The Unsri Research Digital Repository is a centralized platform developed for Universitas Sriwijaya to store, manage, and provide structured access to research documents such as theses, dissertations, academic papers, and publications.

## Key Features

- **Structured Document Storage**: Organized storage based on categories, faculties, and research types.
- **Advanced Search & Filtering**: Fast search by title, author, year, or keywords.
- **Metadata Management**: Each document includes complete metadata for efficient indexing.
- **Role-Based Uploading**: Upload and document management permissions based on user roles.
- **Responsive UI**: A user-friendly interface accessible to students, lecturers, and administrators.

## Technical Details

- **Backend**: Laravel framework with Blade templating engine.
- **Database**: MySQL schema designed to support comprehensive metadata.
- **Frontend**: Lightweight, responsive interface built with Bootstrap.
- **Security**: Laravel middleware for authentication and access control.
- **File Handling**: Document upload and storage system with strict validation.

## Challenges & Solutions

- **High Document Volume**: Optimized queries and metadata indexing to maintain search performance.
- **Metadata Consistency**: Validation mechanisms implemented to enforce uniform metadata entry.
- **Varied User Roles**: Simplified RBAC system to ensure controlled access for different user groups.

## Outcome

The platform enhances accessibility and organization of research publications at Universitas Sriwijaya, improving document retrieval and management across academic departments.
