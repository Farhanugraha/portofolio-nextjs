# Overview

BSP Internal System is a centralized platform developed for Bumi Sawit Indo Permai to streamline monitoring, validation, and management of operational data across departments.

## Key Features

- **Operational Data Monitoring**: Real-time tracking of key plantation and processing metrics.
- **Data Validation Tools**: Ensures data accuracy before submission and processing.
- **Role-Based Access**: Different access levels for supervisors, admins, and field teams.
- **Reporting Dashboard**: Visual summaries for oversight and decision-making.

## Technical Details

- **Backend**: PHP procedural with structured modules for data processing.
- **Database**: MySQL schema optimized for operational records and daily updates.
- **Frontend**: Bootstrap-based UI for lightweight and responsive interaction.
- **Security**: Session-based authentication with data-level access filtering.

## Challenges & Solutions

- **Data Inconsistency**: Added validation steps before submission to reduce errors.
- **Limited Connectivity**: Lightweight queries and caching for low-bandwidth environments.
- **User Training Gap**: UI simplified for field workers with minimal digital experience.

## Outcome

The system improved oversight, reduced data discrepancies, and provided clearer visibility into daily operational activities at Bumi Sawit Indo Permai.
